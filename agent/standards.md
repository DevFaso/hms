# standards.md
**Code Quality:** Write clear code; descriptive names; no magic values; minimize side effects.  
**Error Handling:** Fail clearly; log context (no secrets); validate all inputs.  
**Security:** Never hardcode secrets; sanitize all inputs; enforce auth on every endpoint; apply least privilege.  
**Performance:** Optimize only when needed; prevent N+1 queries; use caching or indexes; avoid unnecessary re-renders.
