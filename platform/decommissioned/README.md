# Decommissioned Components

This directory contains platform components that have been superseded by native implementations.

## patient-mobile-app (Capacitor/Vite Hybrid)

**Decommissioned:** June 2025  
**Replaced by:**
- `patient-ios-app/` — Native SwiftUI iOS app
- `patient-android-app/` — Native Kotlin/Jetpack Compose Android app

**Reason:**  
The hybrid Capacitor-based mobile app has been replaced with native iOS and Android applications to provide better performance, platform-specific UX, and access to native APIs (biometrics, push notifications, offline storage).

**Note:** This code is preserved for reference only. Do not build or deploy from this directory.
